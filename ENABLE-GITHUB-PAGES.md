# 🔧 Enable GitHub Pages - Step by Step

## 📋 Current Issue
The 404 error means GitHub Pages isn't enabled yet for your repository.

## 🚀 Fix It Now:

### Step 1: Go to Repository Settings
Visit: https://github.com/razan-aboushi/razan-aboushi-portfolio/settings/pages

### Step 2: Configure GitHub Pages
1. **Build and deployment** section
2. **Source**: Select "Deploy from a branch"
3. **Branch**: Select "main"
4. **Folder**: Select "/ (root)"
5. Click **"Save"**

### Step 3: Wait for Deployment
- GitHub Pages will deploy in 1-2 minutes
- You'll see a green checkmark when ready
- URL will be: https://razan-aboushi.github.io/razan-aboushi-portfolio

## 🌐 Alternative: Vercel (Recommended & Faster)

### Step 1: Go to Vercel
Visit: https://vercel.com

### Step 2: Import Repository
1. Click "Add New..." → "Project"
2. Connect your GitHub account
3. Select "razan-aboushi-portfolio"
4. Click "Import"

### Step 3: Deploy
1. Vercel auto-detects Next.js
2. Click "Deploy"
3. Live in 2-3 minutes at: https://razan-aboushi-portfolio.vercel.app
